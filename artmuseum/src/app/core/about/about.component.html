Welcome to ArtMuseum!

ArtMuseum is an Angular application that I designed and developed to showcase my skills in Angular development. My goal is to provide a seamless user experience while demonstrating my ability to implement various Angular features.

Key Features:

  • RESTful API integration to perform CRUD operations on a database of items
  • Utilization of ng bootstrap to enhance user experience and provide responsive design
  • Implementation of Angular routing for a seamless user experience
  • Utilization of Reactive Forms to handle user input and validation

I am a highly skilled Angular developer who is passionate about providing the best user experience possible. I have carefully designed ArtMuseum to showcase my skills in a visually appealing and easy-to-use manner. I believe that my implementation of Angular features is unique and interesting. I am confident that users will find ArtMuseum to be a valuable tool for exploring and learning about various pieces of artwork.
